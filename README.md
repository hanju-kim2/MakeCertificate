# MakeCertificate
---
* 성적서 전산화 프로그램
* Raw data(xlsx) + 이미지(jpg)을 알맞은 형태로 조합해 Result Form(xlsx)에 넣어 성적서를 만듦.
---
* 사용 방법
  * MainWindow에서 아이디, 비밀번호를 입력해 성적서 전산화 Window로 이동
  * 모듈, 폴더경로를 입력하고 출력되는 회사 정보 확인
  * 출력될 파일 이름, 성적서 번호, 날짜를 입력하고 만들기 버튼 클릭
  * 결과 경로로 가서 결과 성적서 확인
---
* 사용 가능한 Excel 버전
  * 2016
  * 2019
* 사용 불가능한 Excel 버전
  * 2010
